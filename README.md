<b>Youtube Shorts Text Removal</b>

This script serves to remove text and icons that overlay Youtube Shorts such as:

1. Channel icon and name
2. Music icon and name
3. Captions
4. Search icon that overlays the Short

Overall, this script will deliver a smoother viewing experience on desktop.
